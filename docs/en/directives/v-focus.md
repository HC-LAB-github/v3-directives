# v-focus

## Function Introduction

Input box autofocus

## Example

<input
  :style="{
    display: 'flex',
    justifyContent: 'center',
    alignItems: 'center',
    border: '1px solid #ccc',
    padding: '5px',
    borderRadius: '5px'
    }" v-focus />

## Usage

```typescript {2}
<template>
  <input v-focus />
</template>
```
